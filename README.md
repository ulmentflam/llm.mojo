# LLM.mojo
LLMs in mojo with out the need for Pytorch or cPython with. Inspired by karpathy's (llm.c)[https://github.com/karpathy/llm.c] with a focus on proving out the viability of AutoGrad in pure mojo syntaxt. The focus is to reproduce GPT-2 and GPT-3 along with a parallel PyTorch reference in `train_gpt*.py`. A personal goal of this project for me is to write all code without using any LSPs or LLMs, writing all algorithms (forward and backpropogation) from scratch. I recived feedback recently that my "coding and math expertise" are not strong enough. My goal in builing out this framework is to strenghten those skills. Just like writing a compiler, writing out the fundimentals of genrative models from scratch will strenghten engineering skills and mathmatics. As part of my goal, I will be leverageing Nvidia Nsight and Perfetto for performance analysis and comparison againts the PyTorch implementation of GPT-2. While this project evolves, I will include benchmarking results and other insights into the performance comparitives between Mojo, PyTorch, and even karpathy's C implementation.
